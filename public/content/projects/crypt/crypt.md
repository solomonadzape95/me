## stack
next.js, react 19, typescript, stellar sdk, freighter api, supabase, tailwind v4, framer motion

## about
crypt is a stellar-network wallet and token client. it talks to the freighter wallet via its api for signing and uses supabase as a thin metadata layer for off-chain state. the focus is on a fast, opinionated transaction flow rather than feature-completeness — a small surface that does the common thing well.
